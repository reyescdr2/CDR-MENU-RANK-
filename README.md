# 📚 **CDR Addon Pack - Documentación Oficial**

¡Bienvenido al sistema **CDR (Commands, Design & Ranks)**! Este pack es la herramienta definitiva para la gestión avanzada de servidores y mundos técnicos en Minecraft Bedrock, ofreciendo un control total sin sacrificar la posibilidad de obtener logros.

## 🛠️ **Arquitectura del Sistema**

El proyecto se divide en tres pilares fundamentales que trabajan en perfecta sincronía:

1.  **CDR Menu BP (Behavior):** El motor lógico. Controla la economía infinita, la protección de dimensiones y el sistema de misiones.
2.  **CDR Ranks BP (Ranks):** El sistema de comunicación. Gestiona el chat dinámico y los prefijos de colores (Requiere API Beta).
3.  **CDR Menu RP (Resources):** La cara visual. Contiene todos los iconos premium, texturas personalizadas y traducciones.

## 🚀 **Características Destacadas**

### 💰 **Economía e Infinity Stats**
Nuestro motor de estadísticas virtual permite manejar cifras que superan el límite estándar de Minecraft (32-bit), permitiendo economías verdaderamente globales.
*   **Recompensas Dinámicas:** Pagos automáticos por eliminar mobs y jefes.
*   **Sistema de Corrupción:** Un estado alterado que afecta la economía y añade riesgo al juego.

  <img src="URL_IMAGEN_MENU_ECONOMIA" alt="Economy Menu" width="400">

### 📋 **Sistema de Misiones Trilingüe**
Misiones de caza y minería equilibradas para mantener a los jugadores activos, con soporte total para **Español, Inglés y Portugués**.
*   **Niveles:** Fácil, Normal, Difícil y Hardcore.

### 🛡️ **Seguridad y Dimensiones**
Control absoluto sobre el acceso al **Nether** y al **The End**.
*   **Bypass Inteligente:** Los administradores pueden ignorar estas restricciones.
*   **Rechazo Suave:** Los jugadores son repelidos físicamente si intentan entrar a una dimensión bloqueada.

## 🕵️‍♂️ **HERRAMIENTAS ESTRATÉGICAS (Admin Supremo)**

Esta sección contiene las funciones de alto nivel diseñadas para el control absoluto del entorno.

### ⚡ **Panel de Control Superior**
Acceso a configuraciones que cambian las reglas del servidor en tiempo real.
*   **Gestión de Rangos:** Asignación de permisos incluso para otros rangos Supreme.
*   **Editor de Stats:** Modificación directa de saldos y estadísticas de cualquier jugador.
*   **Precios Globales:** Control total sobre el mercado y multiplicadores de recompensas.

<div align="center">
  <img src="URL_IMAGEN_MENU_SUPREMO" alt="Supreme Menu" width="400">
</div>

### 🔐 **Protocolos de Elevación y Emergencia**
*   **Elevación de Admin a Supreme:** Si un usuario con rango **Admin** (`ADMIND`) necesita acceso a las herramientas de nivel **Supreme**, puede obtenerlo temporalmente mediante el sistema de validación de objetos. 
*   **Acceso por Ítem Secreto:** Al renombrar un objeto a la **Palabra Clave Específica** (conocida por el equipo de desarrollo) y sostenerlo en la mano, se desbloquearán instantáneamente los botones ocultos del **Menú Supremo**. Este método permite que la administración recupere el control total o eleve sus privilegios de forma discreta y segura sin necesidad de modificar archivos internos.

### 💻 **Terminal de Comandos**
Ejecuta cualquier comando directamente desde el menú, diseñado para una administración rápida y eficiente.
